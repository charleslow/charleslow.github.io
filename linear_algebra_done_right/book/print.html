Definition 1.1 Complex Numbers C

A complex number is an ordered pair (a,b), where a,b∈R, but usually written as a+bi.

  • The set of all complex numbers is denoted as C:

C={a+bi:a,b∈R}

  • Addition and multiplication on C are defined by: (a+bi)+(c+di)(a+bi)(c+di)​=(a+c)+(b+d)i=(ac−bd)+(ad+bc)i​ Where a,b,c,d∈R.

Note that R⊂C, since R={a+bi:b=0,a∈R}.
